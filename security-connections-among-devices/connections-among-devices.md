# Connections among Devices

When ***Smart Device***, Smartphone, Smart TV try to connect to another Device, then we should provide security to that Connection. 

***Smart Device*** should have a list of Devices which we already set up before. This way, we know what Device we are connecting to.

For example. Our ***Smart Device*** holds Contacts who we know about and we trust too. The same scenario happens with the last paragraph explained.

Devices, we granted to connect to, from our ***Smart Device***, should be a manual setting. In case, we can do it automatically, it may lead to untrusted Connections.
