# How to connect securely

When someone connects remotely either to a hardware or smart device. It is no secure because nowadays, smart device can connect with other devices remotely. 

Someone can connect to a device using our smart advice. This connection between different hardware or smart device, is dangerous because if this connection is transmitting data to another device which is not ours. I mean, it is a smart device that we don't know where it comes from. Then, this connection kind could lead to real issues.

At the moment, someone connect to another device which has our same Signature. For example, we may use an Account as Signature, like Samsung or Google. We can use an Account to start using our smart device. Then, we might review, if we are connecting with same Signature (Account). 

Now, we connect to, with something that we know which are connecting to. 

Besides, it should be told as well. Someone may connect remotely using a Service. For example, it could be NFC, Wi-Fi, or another one Service that smart device provides nowadays.
