## Trading Analysis according to patterns

The moment for selling the action, you owed. It is when you understand and catch the pattern which is repeating itself consecutively.

The following pattern is going to be, more or less, very similar.

If that action value keeps going higher, in terms of money. Then, you will know, when it is the most appropriate moment to sell that action, you previously purchased.

Besides doing manually trading-analysis. You may create a Software-Application using programming to make it more dynamically and automatically.
