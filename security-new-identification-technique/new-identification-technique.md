# New identification technique

We can consider a vulnerability when Android adds new features. 

User identification process should be reconsidered.

I suggest, there could be another way to identify the user and putting aside the traditional identification techniques like fingerprints recognition, and so on. 

These traditional identification techniques are becoming common to use. 

Typically, we cannot use the same voice all the time. Even though, at times, if we are bored, we use a different voice. 

However, we are going to use the same root voice. Even, if the voice is becoming different in different situations.

So, we might create a new algorithm which can do that. It could be something more sophisticated to identify the user and we protect the person who is allowed to use that device. Then, this could be more difficult to hack or mimic.

Unfortunately, it is becoming more usual to identify the user and at same time more insecure and less sophisticated.
