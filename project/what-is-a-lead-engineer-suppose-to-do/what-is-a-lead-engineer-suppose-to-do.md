## What is a lead engineer suppose to do?

As a lead engineer you need to assign different tasks to each member team according to his/her skills set.

Communication with team should be clear from the start of development of project. 

In order to accomplish this, you can consider these following items.

+ Have daily meetings with team so that each member team can say what he/she is working on, he/she is blocking with, he/she is going to do next day.

+ Write down the tasks/subtasks which should be done to finish the project.

+ Avoid unnecessary meetings and/or irrelevant tasks.

+ Communicate differently with each member team according to his/her tech skills and soft skills.

+ Put into a diagram what you need to ask the team using any design tool like Gimp, Paint, MS Visio, Pencil, Photoshop.
